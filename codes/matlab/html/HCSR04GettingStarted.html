Getting Started with HC-SR04 Add-On Library for Arduino

Before you can use this add-on library in MATLAB®, you need to install the HC-SR04 Arduino® library. To install the HC-SR04 Arduino library,

  1. Download the zip file from GitHub
  2. Unpackage the zip file into a local directory and rename the folder to Ultrasonic
  3. Move the Ultrasonic folder into the libraries folder inside your Arduino sketchbook folder:
  • On Windows®, the default path is "Documents\Arduino\libraries\"
  • On Mac®, the default path is "~/Documents/Arduino/libraries/"
  • On Linux®, the default path is "/home/<username>/Arduino/libraries"


Published with MATLAB® R2016a
