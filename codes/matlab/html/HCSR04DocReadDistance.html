readDistance

Initiate ranging operation and measure distance to nearest object in meters

  • Description
  • Input Arguments
  • Output Arguments
  • Examples
  • See Also

Syntax

dist = readDistance(sensor)

Description

dist = readDistance(sensor) measures the distance to the nearest object, in meters.

Input Arguments

sensor - HC-SR04 sensor connection, specified as an HCSR04 object.

Output Arguments

dist - Measured distance, in meters.

Examples

Measure Distance to Nearest Object Using HC-SR04

Create an Arduino® object, and create the HCSR04 object.

a = arduino('COM3','Uno','libraries','JRodrigoTech/HCSR04');
sensor = addon(a,'JRodrigoTech/HCSR04','D9','D2');

Measure the distance in meters.

dist = readDistance(sensor)

dist =

    0.5100
