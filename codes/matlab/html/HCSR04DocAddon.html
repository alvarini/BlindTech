Syntax

sensor = addon(a,'JRodrigoTech/HCSR04',triggerPin,echoPin)

Description

sensor = addon(a,'JRodrigoTech/HCSR04',triggerPin,echoPin) creates the connection to an HC-SR04 sensor.

Input Arguments

a - Arduino® hardware connection, specified as an arduino object.

'JRodrigoTech/HCSR04' - HC-SR04 sensor library name.

triggerPin - Arduino pin number to start ranging operation of device, specified as a character vector.

echoPin - Arduino pin number to receive the echoed back signal, specified as a character vector.

Properties

HCSR04 object consists of the read-only property Pins .

Pins

The Arduino pins used for communication with the HC-SR04 sensor.

Output Arguments

sensor - HC-SR04 sensor connection, returned as a HCSR04 object.

Examples

Create an HC-SR04 Sensor Connection

Create an Arduino object, and create the HCSR04 object.

a = arduino('COM3','Uno','libraries','JRodrigoTech/HCSR04');
sensor = addon(a,'JRodrigoTech/HCSR04','D9','D2')
